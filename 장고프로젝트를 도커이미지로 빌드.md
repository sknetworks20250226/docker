# 1. 도커 이미지 빌드 및 푸시
    - Docker Hub 계정생
    - 이미지 빌드
    ```
    docker build -t <dockerhub-username>/my-django-app:latest .
    ```
    - 로그인 후 푸시
    ```
    docker login
    docker push <dockerhub-username>/my-django-app:latest
    ```
